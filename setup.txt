
        -------------------------------------
        | Doctor Appointment Booking System |
        -------------------------------------

Frontend :

  - cd doctor-appointment-frontend

  - npm install
  - npm run dev

Backend :

  - cd doctor-appointment-backend

  - python -m venv env
  - source env/bin/activate  (Linux/Mac)
  - .\env\Scripts\activate     (Windows)

  - python -m pip install --upgrade pip
  - pip install -r requirements.txt
      ----- or -----
    python -m pip install -r requirements.txt

  - python run.py

APIs :

  Map([

GitHub / Git Command Reference :

    = To add updated files to git and push to GitHub :

        - git init
        - git add .
        - git commit -m "Initial commit"
        - git branch -M main
        - git remote add origin
        - git push -u origin main

    = To remove a file from git and push the changes to GitHub :

        - git rm filename
        - git commit -m "Removed file"
        - git push origin main

    = If you want to remove it from the commit but keep it in your local folder :
    
        - git rm --cached filename
        - git commit -m "Removed file"
        - git push origin main

    = To check the status of your git repository :

        - git status

    = To check the commit history :

        - git log

    = To check the commit history with a graph :

        - git log --graph --oneline

    = To check the commit history in a single line format :

        - git log --oneline

    = To check the commit history with a graph and all details :

        - git log --graph --all

    = To check the commit history with a graph and all details and show the changes in each commit :

        - git log --graph --all -p

    = To check the commit history with a graph and all details and show the changes in each commit and show the files changed in each commit :

        - git log --graph --all -p --stat

    = To check the commit history with a graph and all details and show the changes in each commit and show the files changed in each commit and show the author of each commit :

        - git log --graph --all -p --stat --author

    = To check the commit history with a graph and all details and show the changes in each commit and show the files changed in each commit and show the author of each commit and show the date of each commit :

        - git log --graph --all -p --stat --author --date

    = To check the commit history with a graph and all details and show the changes in each commit and show the files changed in each commit and show the author of each commit and show the date of each commit and show the commit message of each commit :

        - git log --graph --all -p --stat --author --date --pretty=format:"%h - %an, %ar : %s"

    = To check the commit history with a graph and all details and show the changes in each commit and show the files changed in each commit and show the author of each commit and show the date of each commit and show the commit message of each commit and show the branch of each commit :

        - git log --graph --all -p --stat --author --date --pretty=format:"%h - %an, %ar : %s (%d)"

    = To check the commit history with a graph and all details and show the changes in each commit and show the files changed in each commit and show the author of each commit and show the date of each commit and show the commit message of each commit and show the branch of each commit and show the email of each commit :

        - git log --graph --all -p --stat --author --date --pretty=format:"%h - %an <%ae>, %ar : %s (%d)"

    = To check the commit history with a graph and all details and show the changes in each commit and show the files changed in each commit and show the author of each commit and show the date of each commit and show the commit message of each commit and show the branch of each commit and show the email of each commit and show the hash of each commit :

        - git log --graph --all -p --stat --author --date --pretty=format:"%h - %an <%ae>, %ar : %s (%d) [%H]"

    = To delete a branch both locally and remotely :

        - git branch -d branch_name
        - git push origin --delete branch_name

    = To create a new branch and switch to it :

        - git checkout -b branch_name

    = To switch to an existing branch :

        - git checkout branch_name

    = To merge a branch into the current branch :

        - git merge branch_name

    = To clear git bash history (use with caution) :

        - history -c

    = To clear git bash console (use with caution) :

        - clear
